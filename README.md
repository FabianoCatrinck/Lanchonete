# Lanchonete
Minha Lanchonete

o Instruções para instalação e execução da aplicação:
Baixar o fonte de modo que a pasta Lanchonete fique na raiz da unidade de disco, por exemplo: c:\lanchonete ou d:\Lanchonete...

o Explicação das escolhas técnicas e de design. 
Foquei em ser objetivo indo direto ao ponto dos requisitos sem me preocupar demais com aparência ou usabilidade pro conta do pouco tempo que tive para desenvolver a aplicação.
Usei o delphi community na sua versão mais nova e as imagens foram usadas da aplicação icon8 que é gratuita.
Toda a aplicação foi desenvolvida com componentes nativos.
Criei uma documentação visual simples mas que demonstra que antes do código eu pensei a aplicação e nas suas necessidades.
Usei o SQLite Expert Professional na sua versão trial para desenvolver mais rapidamente a parte do banco de dados.
Neste projeto eu usei várias práticas consideradas boas a nível de mercado, tais como, orientação a objetos, clean code e solid. 
Todas as rotinas possuem verbo no infinitivo como boa prática no entendimento das ações que são executadas.
Nomes claros tanto das rotinas quanto das telas e componentes.
Comandos SQL nas operações de banco de dados, demonstrando minimamente meu conhecimento a respeito de criação de consultas e formulação de respostas a requisitos.

o Eventuais desafios enfrentados e/ou considerações.
A parte mais desafiadora desse projeto foi combinar as regras de negócio na parte de pedidos.
Vi oportunidades de melhora em alguns pontos a nível de usabilidade mas dei prioridade em entregar o projeto.
